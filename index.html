How to play

  1. Tap an arrow to send it flying in the direction it points.
  2. It can only leave if nothing blocks its path to the edge.
  3. Tapping a blocked arrow costs a life.
  4. Clear all arrows to win. Every puzzle is solvable.

Tip: use Hint if you're stuck. Pinch or scroll to zoom on big puzzles.
